# node-hello-world

学习《node 与 express 开发》时的 code

## 添加 nodemon

一个监控工具，在它发现`JavaScript`被修改后会自动重启服务器。

## 启动命令

`node index.js`
`npx nodemon index.js` - 由于`nodemon`并不是安装的全局的，所以要这样启动。具体可查看[nodemon](https://github.com/remy/nodemon)

## 手动重启

`rs`

## jshint

`npx jshint index.js`
